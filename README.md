# 🎄 Santa is Coming to Town — Christmas Countdown

Праздничный таймер обратного отсчета до Рождества с атмосферным музыкальным сопровождением и эффектом падающего снега.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/Merry-Christmas/)

---

### 🌟 Особенности проекта:
* **Эффект снега:** Интеграция библиотеки `Particles.js` для создания динамического фона.
* **Музыкальная атмосфера:** Возможность управления треком (Play/Pause) с использованием кастомных иконок.
* **Точный расчет:** Логика JavaScript автоматически высчитывает оставшееся время до 25 декабря.
* **Доступность:** Использование семантических тегов `header`, `main` и атрибутов `aria-label`.

---

### 🛠 Стек технологий:
* **JavaScript (Vanilla):** Управление аудио-объектами и расчет времени.
* **Particles.js:** Библиотека для создания эффектов частиц (снег).
* **HTML5 & CSS3:** Продвинутая типографика и адаптивная сетка.

---
